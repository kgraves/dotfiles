# dotfiles
Conatins all of my configuration and dotfiles.

### setup
```bash
cd /path/to/git/repos
git clone git@github.com:kgraves/dotfiles.git
cd dotfiles
source setup.sh
```

### management
To update the repo with changes:
```bash
cd /path/to/git/repos/dotfiles
source update.sh
git -a commit
git push
```
